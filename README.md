# תיאור כללי
בפרויקט זה פותחה מערכת מלאה בצד לקוח ובצד שרת, המדמה דף נחיתה לרישום מתעניינים ללימודים באוניברסיטה. המערכת כוללת ממשק ניהול (Admin) לצפייה, ניתוח וייצוא נתוני הנרשמים.

# דפים קיימים במערכת

דף נחיתה ורישום: דף הבית הכולל טופס להזנת פרטי מתעניינים.

דף תודה: דף אישור המוצג למשתמש באופן אוטומטי לאחר שליחת הטופס בהצלחה.

דף התחברות (Login): ממשק כניסה מאובטח למנהל המערכת (Admin).

דף דשבורד לאדמין: הצגת כל הנרשמים בטבלה אינטראקטיבית הכוללת אפשרות לייצוא ל-CSV.

# נתונים נשמרים (MongoDB)
users: שם פרטי, שם משפחה, תחום לימודים, אימייל וטלפון.

admins: שם משתמש וסיסמה מוצפנת.

# נתיבי API (Backend)
POST /submit-user - רישום מתעניין חדש ומחיקת נתונים זמניים.

POST /login - אימות מנהל והנפקת טוקן JWT.

GET /users - שליפת רשימת הנרשמים מהמסד (מוגן בטוקן).

# טיפוסים וסכימות (Schemas)

המערכת משתמשת ב-Schemas של Mongoose כדי להגדיר את מבנה הנתונים:

Admin: כולל שם משתמש (username) וסיסמה מוצפנת (password).

User: כולל את פרטי הנרשם (שם פרטי, שם משפחה, תחום לימודים, אימייל וטלפון).
