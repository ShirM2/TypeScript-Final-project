# מערכת רישום מועמדים - אוניברסיטת אריאל בשומרון

## תיאור הפרויקט
בפרויקט זה פותחה מערכת Full-Stack מלאה הכוללת צד לקוח וצד שרת. המערכת מדמה דף נחיתה לרישום מתעניינים למסלולי לימוד באוניברסיטה, כולל ממשק ניהול (Admin) מאובטח לצפייה, ניתוח וניהול נתוני הנרשמים.

---

## טכנולוגיות בשימוש

### Frontend:
* **React** (עם TypeScript)
* **Vite** - ככלי הרצה ופיתוח.
* **Tailwind CSS** - לעיצוב רספונסיבי ומודרני (כולל שימוש באפקטים של טשטוש ו-Glassmorphism).
* **Radix UI** - לשימוש ברכיבי ממשק נגישים (Form, Select).
* **Axios** - לביצוע בקשות HTTP לשרת.
* **React Router Dom** - לניהול הניווט בין הדפים.

### Backend:
* **Node.js** עם **Express**.
* **TypeScript** - לכתיבת קוד בטוח ומסודר.
* **MongoDB & Mongoose** - לניהול מסד הנתונים והסכימות.
* **JWT (JSON Web Token)** - לאימות מאובטח של מנהל המערכת.
* **Bcrypt** - להצפנת סיסמאות ה-Admin.
* **Dotenv** - לניהול משתני סביבה.

---

## דפים קיימים במערכת
* **דף נחיתה ורישום:** דף הבית הכולל טופס מעוצב להזנת פרטי מתעניינים עם רקע מותאם של האוניברסיטה.
* **דף תודה:** דף אישור המוצג למשתמש באופן אוטומטי לאחר שליחת הטופס בהצלחה.
* **דף התחברות (Login):** ממשק כניסה מאובטח למנהל המערכת.
* **דף דשבורד לאדמין:** הצגת כל הנרשמים בטבלה אינטראקטיבית הכוללת אפשרות לייצוא ל-CSV.

## הוראות הרצה (Installation & Setup)

### 1. צד שרת (Backend)
1. היכנסי לתיקיית `server`.
2. התקיני את הספריות הדרושות:
   ```bash
   npm install
3. הריצי את הפרויקט:
   ```bash
   npm run dev
   ```

---

## נתונים נשמרים (MongoDB)
* **User:** שם פרטי, שם משפחה, תחום לימודים, אימייל וטלפון.
* **Admin:** שם משתמש וסיסמה מוצפנת.

## נתיבי API (Backend)
* `POST /submit` - רישום מתעניין חדש ושמירה במסד הנתונים.
* `POST /admin/login` - אימות מנהל והנפקת טוקן JWT.
* `GET /admin/users` - שליפת רשימת הנרשמים (מוגן בטוקן).
